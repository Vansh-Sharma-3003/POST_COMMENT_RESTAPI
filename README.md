# Project Name : Post Comment RestAPI
<br>

## Overview

<br>
This project implements a robust and scalable commenting system designed to be integrated into any existing web application, such as a blog or content platform. It handles the submission, storage, retrieval, and display of user comments associated with specific posts or articles.

<br>

## Features

<br>
Threaded Comments: Supports nested replies.
<br>
Post Association: Each comment is uniquely linked to a specific content item (post_id).
<br>
User Authentication: Integrates with user data to display author names.
<br>
Architecture: Uses Data Transfer Objects (DTOs) to strictly define the input and output structures for all API interactions.

<br> 

## Technology Stack

<br>
This project is built using the following technologies:
<br>
Language: Java
<br>
Framework: Spring Boot
<br>
Database: MYSQL
